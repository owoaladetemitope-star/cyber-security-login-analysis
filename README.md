# Cybersecurity Login Threat Detection Dashboard

This project analyzes login activity to detect suspicious behavior such as repeated failed login attempts, risky IP addresses, and abnormal login patterns.

## Features
- Detects suspicious IP addresses based on failed login attempts
- Identifies high-risk users
- Assigns risk levels (Low, Medium, High)
- Analyzes failed logins by hour
- Interactive dashboard built with Streamlit

## Tools Used
- Python (pandas, matplotlib)
- Streamlit (dashboard)
- Data analysis techniques

## How to Run
py -m pip install streamlit  
py -m streamlit run app.py
